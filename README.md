# Discover LA: Our Landmarks Project

A fun, visual-first website built by **Carlthorp School 3rd graders** in Santa Monica to explore the history and stories behind famous Los Angeles landmarks.

![In-N-Out Burger — one of 16 LA landmarks featured on the site](https://upload.wikimedia.org/wikipedia/commons/thumb/0/00/In-N-Out_Burger_at_Fisherman%27s_Wharf_in_San_Francisco_%28TK1%29.JPG/960px-In-N-Out_Burger_at_Fisherman%27s_Wharf_in_San_Francisco_%28TK1%29.JPG)

## What Is This?

Each student picks a real LA landmark and learns its story, history, and location. This website is our class hub — click any landmark card to read its story, fun facts, and how far it is from our school. There's also an interactive map to see where everything is!

## Featured Landmarks

| Category | Landmarks |
|----------|-----------|
| **Food** | In-N-Out Burger, Randy's Donuts, Original Farmers Market |
| **Culture** | Santa Monica Pier, Hollywood Bowl, Watts Towers, Capitol Records, Griffith Observatory |
| **Nature** | La Brea Tar Pits, Venice Canals |
| **History** | Angels Flight, Olvera Street, Union Station, LA Coliseum, Dodger Stadium, Bradbury Building |

## Tech Stack

- [Vite](https://vitejs.dev/) + [React](https://react.dev/) + TypeScript
- [Tailwind CSS](https://tailwindcss.com/) v4
- [Leaflet](https://leafletjs.com/) interactive map with [CARTO](https://carto.com/) light tiles
- Images from [Wikimedia Commons](https://commons.wikimedia.org/)
- Deployed to GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```

Open [localhost:5173](http://localhost:5173) to view it locally.
